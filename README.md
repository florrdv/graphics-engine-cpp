# Graphics-engine-cpp
### A *(soon to be)* 3D graphics engine written from the ground up in C++

# Implement features
- Basic shapes
  - Diamond
  - Quarter circle
  - Chessboard
  - Eye
- L-systems
  - Basic L-systems
  - L-systems with brackets
  - Stochastic L-systems [stochastic-example.L2D](stochastic-example.L2D)
- 3D Lijntekeningen
- 3D Lichamen
- 3D L-Systemen
- Z-Buffering met lijnen
